## 📌 TIPS NGECHAT BIAR EFEKTIF

1. **Tag orang spesifik** kalau action item-nya untuk mereka (`@Dhevina`, `@Syendhi`)
2. **Bullet point** bukan paragraf — orang malas baca paragraf
3. **Link langsung** ke halaman web atau file — jangan suruh navigate
4. **Action item di akhir** — jelas apa yang harus mereka lakukan
5. **Confirm di chat** — minta mereka reply "ok" atau emoji 👍 supaya kamu tau mereka udah baca
