# PROJECT MASTER: KKN UKDW STEM 2026

## Vision & Mission
- **Why:** Menyelesaikan program KKN Tematik International Service Learning STEM (Kolaborasi UKDW × Hong Kong PolyU) dengan sukses, berdampak nyata bagi target (SD/SMP dan SMA), serta mendokumentasikan prosesnya secara rapi untuk keperluan laporan dan berpotensi sebagai data skripsi.
- **Fokus Utama:** 
  1. **SO Eyecare:** Skrining mata (Vision Screening) & Edukasi untuk SD/SMP.
  2. **Workshop Softskill:** Program inovatif dan anti-mainstream untuk SMA.
