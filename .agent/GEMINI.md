# Instruksi Workspace Khusus KKN (Persisten)

Instruksi ini secara otomatis dimuat oleh Gemini CLI untuk menjaga konsistensi alur kerja di dalam proyek KKN ini.

## Aturan Utama (MANDATORY)
1. **Pusat Memori & Progres:** Setiap kali pengguna (user) meminta untuk mencatat, menyimpan progres, atau memperbarui hal-hal terkait KKN, kamu **WAJIB** melakukannya dengan memodifikasi file `_Dex-Brain/Dex-Core.md`. 
2. **Larangan:** **DILARANG KERAS** membuat file `Dex-Core` yang baru di folder lain. Jika pengguna menyuruh "update ke Dex-Core", selalu rujuk file yang berada di `_Dex-Brain/Dex-Core.md`.
3. **Rencana Kerja:** Jika butuh melihat rencana besar, rujuk file `_Dex-Brain/Rencana-Kerja.md`.

Dengan adanya file `GEMINI.md` ini, perilaku AI/Gemini CLI akan terekam secara persisten untuk sesi ini dan sesi-sesi berikutnya di workspace ini.
