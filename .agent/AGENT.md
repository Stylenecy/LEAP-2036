# Instruksi Lokal Agent: Dex OS (Proyek KKN STEM 2026)

## Tujuan
Memastikan Gemini AI selalu selaras dengan aturan *Dex OS* dan konteks khusus proyek KKN STEM 2026.

## Aturan Utama (MANDATORY)
1. **Pusat Memori & Progres (Root Hub):** Setiap pencatatan, penyimpanan progres, atau pembaruan status lintas proyek harus dilaporkan ke `D:\AT Kuliah\All of Project\_Dex-Brain\Dex-Core.md`. 
2. **Pusat Memori Lokal:** Pembaruan status atau detail khusus proyek ini (KKN) dilakukan di `.agent/Dex-Core.md`.
3. **Struktur Dex OS:** Patuhi struktur `/docs`, `/assets`, `/src`, dan `/.agent`.
4. **Larangan:** **DILARANG KERAS** menghapus data/histori saat merapikan file. Jika file terlalu panjang, pecah menjadi beberapa file.
5. **No BS:** Komunikasi harus presisi, logis, *straight to the point*, dan bertindak sebagai *Thinking Partner* (Dex 2.0).

## Konvensi
- **Proyek:** KKN Tematik International Service Learning STEM 2026.
- **Stack (src/kkn-hub):** Vanilla JS, HTML, CSS, Fuse.js, Google Gemini API, di-*deploy* ke Vercel.
